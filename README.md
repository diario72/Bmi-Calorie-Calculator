# BMI & Calorie Calculator

This is a Python-based application that calculates BMI, BMR, and daily calorie intake based on user goals.

## Features
- BMI and calorie calculation
- Input validation
- PostgreSQL database integration

## Technologies
- Python
- PostgreSQL
- psycopg2

## How to run
1. Install dependencies
2. Configure database
3. Run main.py

## Requirements
pip install -r requirements.txt
