# BMI & Calorie Calculator

This is a Python-based application that calculates BMI, BMR, and daily calorie intake based on user goals such as weight loss, maintenance, or gain.

## Features

* BMI and calorie calculation
* Input validation and error handling
* PostgreSQL database integration for storing user data

## Technologies

* Python
* PostgreSQL
* SQL
* psycopg2

## How to Run

1. Clone the repository
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Configure your PostgreSQL database connection in `main.py`
4. Run the program:

   ```
   python main.py
   ```

## Requirements

* Python 3.x
* PostgreSQL
